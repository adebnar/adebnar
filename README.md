## Hi, I'm Andrew 👋

I build **open-source AI-agent and automation tooling**. Right now my main focus is a native Android app for the Hermes agent gateway — taking a powerful self-hosted agent and making it usable from your phone.

### 🔭 Featured project — [Hermes for Android](https://github.com/adebnar/hermes-android)
A native **Kotlin + Jetpack Compose (Material 3)** client for the Hermes agent gateway. Connect to your self-hosted gateway over your private network and get the full surface from your phone:

- 💬 **Chat** — streaming responses, model picker, slash-command palette, `@` file mentions, image attachments, tool-call approvals
- 🗂️ **Sessions** grouped by workspace, with pinning
- 🧠 **Models & profiles** switchable on the fly
- ⏰ **Cron** — create, edit, pause/resume, run-now scheduled jobs
- 📊 **Usage** analytics — daily token charts, per-model breakdown
- 🔌 **MCP servers, API keys & settings** managed live against the gateway

*Built with MVVM + Hilt + OkHttp (REST + WebSocket JSON-RPC).*
